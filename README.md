# Ticket Support System

## 📌 Description

This project is an internal support (helpdesk) system built with Laravel, allowing users to:

- Create support tickets
- Categorize issues
- Assign tickets to agents
- Add comments (internal/public)
- Track status and priority

---

## 🚀 Tech Stack

- Laravel 11+
- PHP 8.2 / 8.3
- MySQL (MariaDB via Docker)
- Docker & Docker Compose
- Adminer (Database UI)
